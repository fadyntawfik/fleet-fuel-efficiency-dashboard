## 📊 Fleet Fuel Efficiency Dashboard

- This project analyzes fleet vehicle performance using Excel to identify inefficiencies and reduce fuel costs.

## 🔧 Tools Used
- Microsoft Excel
- Data Analysis
- Data Visualization
  ## 📈 Key Metrics
- Miles per Gallon (MPG)
- Fuel Cost per Mile
- Total Fuel Cost
  ## 🔍 Key Insights
- Identified high-cost vehicles with low fuel efficiency
- Highlighted opportunities to reduce operational expenses
- Provided data-driven recommendations for fleet optimization
